Basic commands.


If you want to use commands it can be usefull to know what what some basic commands are. So i wil tell you some basic commands to get you started.


TP.

TP is a very simple command to teleport entity's, it is set up in this way: /tp [target selector] [coordinations or target selector. the first target selector specifies wich entity needs to be teleported, the coordinations makes that entity teleport to that cords and the second target selector makes the entity teleport to the entity specified.

Fill.

The fill command is also very simple, it is set up in this way: /fill [coordinations][coordinations] [block]. When you have two coordinations specified it will fill a square or rectangle with the block you specified.

Give.

Give is set up in this way: /give [target selector] [item] [count]. The item can be any item that exists (even ones that aren't in the creative menu) and the count can be any number, but if this munber is higher than 64 it will give multiple stacks to the selected entity.

Gamerule.

Gamerule is the hardest of this list, it is set up in this way: /gamerule [gamerule name] [true/false]. Gamerules are a simple way to change how to game behaves, you can change the day/night cicle or fire damage. Basicly it is a way to change small simple thing from the game, here is a wiki page about gamerules.
